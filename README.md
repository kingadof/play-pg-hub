# play-pg-hub

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放并展示一系列独立开发的 HTML 页面。每个页面均为自包含的静态网页，不依赖外部服务或特定后端。适合用作页面集合的发布与索引。

## 目录结构

```
/
├── index.html          # 仓库首页 / 页面导航
├── page-a/             # 页面 A 相关文件
│   └── index.html
├── page-b/             # 页面 B 相关文件
│   └── index.html
└── ...
```

每个子目录对应一个独立页面，内含该页面的 HTML、CSS、JavaScript 及资源文件。

## 页面归档说明

- 所有页面均为纯前端实现，不涉及后端逻辑。
- 页面之间相互独立，无数据共享或依赖关系。
- 页面内容以信息展示、工具或实验性功能为主。
- 不保证所有页面长期可用，部分页面可能随维护而更新或移除。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交建议或报告问题。
- 不接受直接向仓库提交页面代码的 Pull Request。
- 如有页面引用外部资源（如字体、图标库），请确保其来源稳定且符合许可协议。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或任意子目录下的 `index.html` 文件即可浏览对应页面。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问。

## 许可

本仓库内代码仅供学习与参考，具体页面可能包含独立的许可声明。请以各页面目录内的说明为准。
